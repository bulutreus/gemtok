# Sütun üstü hediye görselleri (WebP)

Bu klasöre **`1.webp`** … **`10.webp`** koyun (sütun 1: 1 ve 2, … sütun 5: 9 ve 10).

Vite `client/App.jsx` üzerinden bu dosyaları derlemeye dahil eder; tarayıcıda doğrudan bu görseller kullanılır.

İsterseniz ek yedek olarak `client/public/assets/` altına da aynı isimlerle dosya koyabilirsiniz.

# Ülke Yarışı Oyunu

TikTok Live tarzı 20 ülke yarış oyunu.

## Çalıştırma

```bash
npx serve .
```

Tarayıcıda açılan adrese git (ör. `http://localhost:3000`).

## TikFinity

1. TikFinity masaüstü uygulamasını aç
2. TikTok LIVE'a bağlan
3. Oyun `ws://127.0.0.1:21213` adresine bağlanır

- `?noConnect=1` — otomatik bağlantıyı kapat
- **Boşluk** — demo modu
- **Şeride tıkla** — test
